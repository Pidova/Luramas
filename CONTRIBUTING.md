# Contributing to Luramas

Thank you for your interest in contributing to the Luramas decompiler framework! 

To keep our development standards, all contribution documentation has been moved to the documentation hub.

### Read the Contribution Guide Here:
 **[Luramas Contribution Guidelines](https://pidova.github.io/Luramas-Docs/docs/contributions/)**


### Quick Reminder
Before submitting a Pull Request, please make sure your local workspace compiles successfully against Luramas:

```
bash
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build .
```
