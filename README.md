# Luramas

**The Retargetable & Customizable Decompiler Framework**

Luramas is an enterprise-grade, retargetable decompiler framework engineered to disassemble, optimize, and lift abstract virtual machine bytecode architectures back into an easily analyzable and recompilable state. 

By simplifying target layout parsing from core semantics, the framework allows you to construct custom frontends (**Lifters**) for arbitrary architectures while using a unified, high-performance optimization passes.

***This repository serves as the framework core for Luramas and contains the engines infrastructure.***
